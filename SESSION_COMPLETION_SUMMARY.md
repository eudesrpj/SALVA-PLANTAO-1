## 🚀 Como Começar Deploy

### Opção Mais Fácil (Render - 10 min)

```
1. Abra: https://render.com
2. Login com GitHub
3. Criar Web Service → escolha este repositório
4. Configure build command: npm ci && npm run build
5. Configure start command: npm run start
6. Adicione environment vars (copy do DEPLOY_QUICK_START.md)
7. Criar PostgreSQL database → automático
8. Deploy → aguarde 2-3 min
9. ✅ App em produção!
```

**Arquivo de referência**: [DEPLOY_QUICK_START.md](./DEPLOY_QUICK_START.md)

## 💡 Pro Tips

### Tip 1: Comece pelo Render
```
Mais fácil, mais rápido, gratuito no tier inicial.
Se depois quiser mudar para GCP = nenhum problema.
```

### Tip 2: Use o Script
```powershell
.\deploy.ps1 -platform render
# Economiza tempo e evita erros
```

### Tip 3: Database Neon
```
Se usar Neon.tech:
- 4 minutos para setup
- Gratuito forever (pequeno volume)
- Integração super fácil
Ver: NEON_QUICKSTART.md
```

### Tip 4: Salve URLs
```
Quando deploy terminar, guarde:
- https://seu-app.render.com
- DATABASE_URL (segredo!)
- Admin URL (se houver)
```

| Erro | Solução |

| `Login não funciona` | JWT_SECRET não configurado |
